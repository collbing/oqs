THE OBJECT QUERY SYSTEM, release 0.1 (17 Jan, 2008)
---------------------------------------------------
http://www.opoo.org


1. INTRODUCTION

Object Query System(OQS)- OQS is a simple object/relational persistence and query service. OQS lets you develop
persistent classes using simple query string (Object Query String, OQS). Just like native SQL, object query 
string is very powerful, it's easy to understand. 


2. RELEASE INFO

The OQS 0.1 requires JDK 1.4.2. Hibernate 3 is required for building the project, but it's optional lib at 
runtime.

To run oqs, you need:
Spring Framework 2.0 (spring core, spring jdbc)
Commons Codec 1.3
Commons Collections 3.2
Commons BeanUtils 1.6
Commons Lang 2.3
Commons Logging 1.0

Hibernate 3.0 **optional**
